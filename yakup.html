Table Etiketleri Ve Örnekleri

Table Etiketi

Web sayfamıza tablo eklemek için kullandığımız etikettir.
Tablolar profesyonel web sayfalarının ayrılmaz öğelerindendir.
Web üzerinde kullanılan tüm tablolar satırlar ve bu satırların içerisindeki sütunlardan oluşur.

Tr Etiketi

Tablonun "satır"ını belirtir.
Açılımı table row'dur.
Dilimizde "tablo satırı" anlamına gelir.

Td Etiketi

Tablonun "sütun"unu belirtir.
Açılımı table data'dır.
Dilimizde "tablo verisi " anlamına gelir.

Thead Etiketi

thead etiketi HTML tablosunun üst içeriğini gruplandımak için kullanılır.

Tbody Etiketi

tbody etiketi HTML tablosunun gövde içeriğini gruplandımak için kullanılır.

Tfoot Etiketi

tfoot etiketi HTML tablosunun alt içeriğini gruplandımak için kullanılır.
